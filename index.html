What is the 3MF Compare Tool?

A free, browser-based utility for 3D printing enthusiasts who use BambuStudio or OrcaSlicer. Drop in two .3mf files and instantly see a side-by-side breakdown of every setting that differs — no software to install, no account required.

Why compare 3MF files?

When troubleshooting print quality or sharing settings on forums like Reddit, it's hard to know exactly what changed between two profiles. Your slicer doesn't offer a diff view. This tool does — showing changed slicer settings, your printer profile, and filament info all in one place. Built for BambuStudio and OrcaSlicer, but also works with Elegoo, other OrcaSlicer forks, and PrusaSlicer-based slicers.

How does it work?

A .3mf file is a ZIP archive. Inside is a project_settings.config JSON file containing all your slicer settings. This tool reads both files in your browser, parses the JSON, and shows only the differences in a searchable, copyable table.

Privacy & Security

  • All processing happens locally in JavaScript
  • Your files are never uploaded to any server
  • No data is transmitted anywhere — ever
  • The only external resource is the JSZip library, loaded with a security hash (SRI) to prevent tampering
  • Open source — inspect the page source any time
